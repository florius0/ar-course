- Сборка приложений под iOS и Android

### II. ARFoundation

#### 2. Трекинг плоскостей

- типы плоскостей

- [Задание 2](#задание-2-трекинг-плоскостей)

#### 3. Трекинг лиц

### III. Продвинутая работа c Unity и ARFoundation

#### 1. Scriptable Objects, AssetBundles, Addressables
